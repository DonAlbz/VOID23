# VOID23
Progetto per CLIL di GPOI

VOID23 INSTALLATION INSTRUCTIONS

1. Install XAMPP.
2. Copy the "void23" folder into C:\xampp\htdocs.
3. Start Apache and MySQL from XAMPP.
4. Open http://localhost/phpmyadmin.
5. Create a database called "void23".
6. Import the void23.sql file.
7. Open the website at http://localhost/void23/.
